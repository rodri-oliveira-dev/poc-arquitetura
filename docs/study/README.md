# Estudos orientados

Este documento organiza uma lista inicial de estudos derivados das personas do repositorio.

## Itens iniciais

1. Matriz funcional de Ledger e Balance.
2. Relatorio local de divergencia da projecao.
3. Auditoria simples para atividades locais de estudo.
4. Metricas locais de baixa cardinalidade.
5. Analise de cobertura orientada a risco.
6. Revisao de contratos de erro e status codes.
