poc-arquitetura

Portal local HTTPS da POC para acessar APIs, Swagger e ferramentas de observabilidade durante o desenvolvimento.

APIs via Nginx

Ledger API Swagger https://ledger.localhost:7443/swagger Balance API Swagger https://balance.localhost:7443/swagger Auth API Swagger https://auth.localhost:7443/swagger

APIs diretas

Ledger API direta http://localhost:5226/index.html Balance API direta http://localhost:5228/index.html Auth API direta http://localhost:5030/index.html

Observabilidade local

Grafana http://localhost:3000 Jaeger http://localhost:16686 Prometheus http://localhost:9090
Nginx e HTTPS local sao opcionais; as APIs continuam disponiveis nas portas HTTP do compose principal.
